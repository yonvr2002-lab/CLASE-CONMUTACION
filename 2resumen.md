## Módulo 3: Reglas de Comunicación en la Red

En este módulo se analizan las reglas que permiten que los dispositivos puedan intercambiar información de manera correcta dentro de una red. Estas reglas son conocidas como protocolos y son fundamentales para cualquier tipo de comunicación digital.

### Temas importantes

**Protocolos de red**  
Son normas que definen cómo se envían y reciben los datos. Algunos de los más conocidos son:
- HTTP / HTTPS (navegación web)
- DNS (traducción de nombres a direcciones IP)
- FTP (transferencia de archivos)

**Modelo OSI**  
Es una forma de dividir el proceso de comunicación en 7 capas, lo que facilita su estudio:
- Aplicación  
- Presentación  
- Sesión  
- Transporte  
- Red  
- Enlace de datos  
- Física  

Cada capa cumple una función específica dentro del envío de información.

**Modelo TCP/IP**  
Es el modelo que se utiliza en la práctica y está compuesto por 4 capas:
- Aplicación  
- Transporte  
- Internet  
- Acceso a la red  

**Encapsulación de datos**  
Se refiere al proceso en el cual la información va pasando por cada capa y se le agrega información adicional para que pueda ser enviada correctamente.

### Análisis personal

Este módulo me ayudó a entender que la comunicación en redes no es algo simple, sino que sigue una estructura organizada que permite que los dispositivos funcionen correctamente sin errores.

## Módulo 4: Conexión y Funcionamiento en la Red

En este módulo se estudia cómo los dispositivos se conectan entre sí dentro de una red local y qué elementos permiten esa conexión.

### Temas importantes

**Dirección MAC**  
Es un identificador único que tiene cada dispositivo. Se utiliza para reconocer equipos dentro de una red local.

**Ethernet**  
Es la tecnología más utilizada en redes LAN. Define la forma en que los dispositivos envían información a través del cableado.

**Switch**  
Es un dispositivo que conecta varios equipos en una red y envía los datos únicamente al destinatario correcto, mejorando el rendimiento.

**Protocolo ARP**  
Permite encontrar la dirección MAC de un dispositivo a partir de su dirección IP.

**Acceso al medio**  
Existen diferentes formas en las que los dispositivos pueden usar la red:
- Por turnos (controlado)  
- Libre acceso con verificación (como Ethernet)

### Análisis personal

Gracias a este módulo comprendí cómo los dispositivos se comunican físicamente y cómo logran identificarse dentro de una red, lo cual es clave para entender redes reales.
